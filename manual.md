# Todo管理システム 操作マニュアル

## 起動方法

### バックエンド
```bash
cd backend
source venv/bin/activate
alembic upgrade head
uvicorn main:app --reload --port 8000
```

### フロントエンド
```bash
cd frontend
npm run dev
```

### アクセスURL

| サービス | URL |
|----------|-----|
| フロントエンド | http://localhost:5173/ |
| バックエンドAPI | http://localhost:8000/ |
| API ドキュメント (Swagger) | http://localhost:8000/docs |

---

## 操作手順

### 1. ユーザー登録（初回のみ）

現在UIに登録画面がないため、ターミナルで実行する：

```bash
curl -X POST http://localhost:8000/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"password123"}'
```

### 2. ログイン

1. http://localhost:5173/ にアクセス → ログイン画面にリダイレクト
2. メールアドレス: `test@example.com`、パスワード: `password123` を入力
3. 「ログイン」ボタンをクリック → ダッシュボードへ遷移

### 3. ダッシュボード確認

- 完了率、今日のタスク数、期限超過タスク数が表示される
- 「タスク追加」ボタンから新規作成が可能

### 4. タスク追加

1. ヘッダーの「タスク一覧」→「タスク追加」ボタンをクリック
2. タイトル（必須）、説明、優先度（緊急/高/中/低）、カテゴリ、期限を入力
3. 「保存」をクリック → タスク一覧に遷移

### 5. タスク一覧で管理

- **フィルタ**: ステータス / 優先度 / カテゴリで絞り込み
- **ソート**: 作成日順 / 期限順 / 優先度順
- **完了切替**: チェックボックスをクリック
- **編集**: タスク名をクリック → 編集画面へ遷移
- **削除**: 「削除」ボタン → ゴミ箱へ移動

### 6. ゴミ箱

- ヘッダーの「ゴミ箱」から削除済みタスクを確認・復元できる
- 30日後に自動で完全削除される

### 7. 設定

- ヘッダーの「設定」からカテゴリの追加・削除が可能
- カテゴリ名と色を指定して作成する
