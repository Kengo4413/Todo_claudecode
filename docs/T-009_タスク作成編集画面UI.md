**ステータス:** 未着手

## 概要
タスクの全項目を入力できるフォーム画面。作成と編集で共用。

## 完了条件
- [ ] フォーム項目: タイトル、説明、期限（日時ピッカー）、優先度（4段階セレクト）、カテゴリ（セレクト）
- [ ] バリデーション（タイトル必須、日本語エラーメッセージ）
- [ ] 3タップ/クリック以内でタスク追加完結
- [ ] タスク入力〜保存を30秒以内（KPI）
- [ ] 編集モード: 既存データのプリフィル
- [ ] 保存成功後にタスク一覧へ遷移
- [ ] レスポンシブ対応（375px〜）
